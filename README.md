# tg: @custom_moviefinder_bot

### Бот запускался локально и сейчас отключён



Асинхронный бот для поиска информации о фильме, принимающий на вход название фильма (возможно, неточное) и возвращающий:
- Название фильма
- Год выпуска
- Постер фильма
- Текстовое описание
- Ссылки на сайты для бесплатного просмотра (фильтруя сайты требующие подписку для просмотра)

Также реализованы команды:
 - /start - стартовое сообщение
 - /help - описание работы бота и доступные команды
 - /history - история поиска фильмов
 - /stats - количественная стастистика поиска фильмов 

Данные по каждому пользователю сохранаются в небольшой базе данных, реализованной на SQL (через sqlite3)

Поиск фильма реализован через Google API, Кинопоиск API и использование регулярных выражений

### aiogram, aiohttp, asyncio, re, sqlite3

Структура кода: 
 - Класс Database в отдельном файле с реализацией запросов
 - Файл links для регулярных выражений и работы со ссылками
 - Файл api для работы с апишками
 - В мейне вся работа с ботом 
 - В конфиге все ключи и сообщения для вывода
